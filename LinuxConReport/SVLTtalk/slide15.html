KVM: the Latest from the Development Team

Chris Wright, Red Hat

  • kvm has replaced xen as the VM integrated directly with the kernel.
  • qemu emulator is heavily reliant on the kvm device driver.
    • Machine-to-machine paradigm for servers rather than user-target in embedded.
  • Support for direct control and communication with host's peripherals, esp. PCI, is being added.
  • Guests can run at 80% native speed although implementation remains buggy!
  • Challenges include guest-host coordination of memory (via Nested Page Tables) and scheduling.
